# Guards

Guards are additional conditions that can be added to function clauses using `when`.

They allow you to specify constraints beyond simple pattern matching.

Guards must be pure expressions that return true or false.

export const defaultCode = `defmodule NumberChecker do
  def check(x) when is_integer(x) and x > 0 do
    "#{x} is a positive integer"
  end

  def check(x) when is_float(x) do
    "#{x} is a float"
  end

  def check(x) when is_binary(x) do
    "#{x} is a string"
  end

  def check(x) do
    "#{x} is something else"
  end
end

IO.puts(NumberChecker.check(42))
IO.puts(NumberChecker.check(3.14))
IO.puts(NumberChecker.check("hello"))
IO.puts(NumberChecker.check(:atom))`
