# Module Attributes

Module attributes in Elixir serve as constants and metadata.

They are defined with `@` and can only be set at compile time.

Common uses include configuration values, documentation, and compile-time constants.

export const defaultCode = `defmodule Constants do
  @pi 3.14159
  @module_name "Constants Module"
  @author "Elixir Developer"

# Attributes can be used in functions

def circle*area(radius) do
@pi * radius \_ radius
end

def get_metadata do
%{
module: @module_name,
author: @author,
pi: @pi
}
end
end

IO.puts("Area of circle with radius 5: #{Constants.circle_area(5)}")
IO.inspect(Constants.get_metadata())`
