# Numbers

As you can see, numbers in Elixir are either integers or floats. You can use `is_integer`, `is_float` and `is_number` to check what a value is.

Elixir allows expressing numbers in decimal, binary, octal, hexadecimal and exponential notation.

Normally, integers can be arbitrairly big and they never overflow (or underflow). However, the AtomVM runtime which we're using in the browser only supports integers up to 64 bits and overflow raises an error. Floats are always stored on 64 bits and overflow raises an error.
