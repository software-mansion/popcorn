# Basic Types

In this chapter we will learn more about Elixir basic types: integers, floats, booleans, atoms, and strings. Let's see the code snippet for some examples of them.

<EditorCode>
```elixir
IO.inspect 42, label: "integer"
IO.inspect 3.14159, label: "float"
IO.inspect true, label: "boolean"
IO.inspect :elixir, label: "atom"
IO.inspect "Hello, Elixir!", label: "string"
```
</EditorCode>
