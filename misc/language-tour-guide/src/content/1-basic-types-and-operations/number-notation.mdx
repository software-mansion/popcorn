# Number notation

Elixir allows expressing numbers in decimal, binary, octal, hexadecimal and exponential notation.

<EditorCode>
```elixir
IO.inspect 123, label: "decimal notation"
IO.inspect 0b1010, label: "binary notation"
IO.inspect 0o777, label: "octal notation"
IO.inspect 0x1F, label: "hexadecimal notation"
# IO.inspect 1.0, label: "decimal notation"
# IO.inspect 1.0e-10, label: "exponential notation"
```
</EditorCode>
