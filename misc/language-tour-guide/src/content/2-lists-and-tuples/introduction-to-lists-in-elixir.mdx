# Introduction to Lists in Elixir

In this chapter we will learn two of the most used collection data-types in Elixir: lists and tuples. Let's start with lists. Elixir uses square brackets to specify a list of values. Values can be of any type.

<EditorCode>
```elixir
# Try changing '3' to 'true'
IO.inspect [1, 2, 3, 4], label: "List of integers"

IO.inspect length([1, 2, 3]), label: "Length of the list"
```
</EditorCode>
