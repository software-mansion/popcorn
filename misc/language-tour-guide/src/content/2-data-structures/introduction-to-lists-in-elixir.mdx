# Introduction to Lists in Elixir

In this chapter we will learn two of the most used collection data-types in Elixir: lists and tuples. Elixir uses square brackets to specify a list of values. Values can be of any type.

<EditorCode>
```elixir
list = [1, 2, true, 3]
IO.inspect(list, label: "List with different types")
IO.inspect(length(list), label: "Length of the list")
```
</EditorCode>
