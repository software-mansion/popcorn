# Modules

Elixir programs are organized into modules.

Modules group related functions together and provide namespacing. Every Elixir file typically defines one or more modules using the `defmodule` macro.

This example uses the `IO` module from Elixir's standard library and the `:rand` module from Erlang.

In Elixir, you can call Erlang modules directly by prefixing them with a colon.

export const defaultCode = `defmodule MyApp do
  def main do
    IO.puts("My favorite number is #{:rand.uniform(10)}")
  end
end

MyApp.main()`
