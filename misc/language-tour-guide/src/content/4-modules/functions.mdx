# Functions

Functions in Elixir are defined inside modules using `def`.

A function can take zero or more parameters. The number of parameters is called the function's arity.

In this example, `add/2` takes two parameters (arity of 2).

Function names in Elixir follow snake_case convention.

export const defaultCode = `defmodule Calculator do
  def add(x, y) do
    x + y
  end
end

IO.puts(Calculator.add(42, 13))`
