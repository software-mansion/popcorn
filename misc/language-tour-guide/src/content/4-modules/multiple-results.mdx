# Multiple Return Values with Tuples

Elixir functions can return multiple values using tuples.

Tuples are ordered collections written with curly braces `{}`.

The pattern `{a, b} = {value1, value2}` destructures the tuple into variables.

export const defaultCode = `defmodule Swapper do
  def swap(x, y) do
    {y, x}
  end
end

{a, b} = Swapper.swap("hello", "world")
IO.puts("#{a} #{b}")`
