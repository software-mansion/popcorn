# Import, Alias, and Use

Elixir provides three directives for working with modules:

- `import` - brings functions from another module into the current scope
- `alias` - creates a shortcut for a module name  
- `use` - invokes macros from another module

You can also call functions with their full module name like `Enum.map/2`.

export const defaultCode = `defmodule MyApp do
  import Enum, only: [map: 2]
  alias String, as: Str
  
  def main do
    result = map([1, 2, 3], fn x -> x * 2 end)
    IO.puts("Doubled: #{inspect(result)}")
    IO.puts("Length: #{Str.length("Hello")}")
  end
end

MyApp.main()`
