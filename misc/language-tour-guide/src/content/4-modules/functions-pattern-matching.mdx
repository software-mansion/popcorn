# Pattern Matching in Functions

Elixir allows you to define multiple clauses for the same function using pattern matching.

The runtime will try each clause in order until it finds a match.

This is a powerful feature that replaces many conditional statements in imperative languages.

export const defaultCode = `defmodule Greeting do
  def hello(:world) do
    "Hello, World!"
  end
  
  def hello(name) when is_binary(name) do
    "Hello, #{name}!"
  end
  
  def hello(number) when is_number(number) do
    "Hello, number #{number}!"
  end
end

IO.puts(Greeting.hello(:world))
IO.puts(Greeting.hello("Elixir"))
IO.puts(Greeting.hello(42))`
